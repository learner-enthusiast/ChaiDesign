Docs chai-* utility reference

Style with
chai classes.

A zero-dependency utility class engine. Add chai-* classes to any element — styles are applied at runtime and classes are removed from the DOM.

example.html 
<div class="chai-flex chai-gap-16 chai-p-24">
  <div class="chai-bg-blue-500 chai-text-white chai-p-12 chai-rounded-8">
    Hello chai
  </div>
</div>
